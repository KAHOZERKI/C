🧠 GeniusIdiot (Console & WinForm)

Интерактивное приложение-тест для оценки «уровня интеллекта» пользователя. Проект демонстрирует разделение ответственности между UI и бизнес-логикой, а также работу с сохранением данных.

🚀 Ключевые технические особенности

В рамках этого проекта я реализовал:

Многослойная архитектура: Вынос бизнес-логики в отдельную библиотеку классов GeniusIdiot.Library (Class Library). Это позволяет использовать одни и те же методы как для консольной версии, так и для Windows Forms.

Работа с данными (I/O): Реализована система хранения результатов тестирования в формате JSON с использованием библиотеки Newtonsoft.Json.

Алгоритмы: Логика случайного выбора вопросов без повторений в рамках одной сессии.

Generic Collections: Активное использование List<T>, IEnumerable и методов расширения LINQ для фильтрации и форматирования вывода результатов.

Валидация данных: Реализованы защитные проверки (Guard Clauses) для пользовательского ввода, предотвращающие падение приложения при некорректных данных.

🛠 Стек технологий

Language: C# 10.0+ / .NET 10.0

UI: Console Interface, WinForms

Data: JSON (Serialization/Deserialization)

Tools: Visual Studio 2026, Git, NuGet (Newtonsoft.Json)

📈 Что проект говорит о моих навыках

SOLID & DRY: Я стремлюсь к чистоте кода, избегая дублирования (например, метод расчета диагноза вынесен в общую библиотеку).

Понимание UX: Программа ведет диалог с пользователем, обрабатывает ошибки ввода и предлагает повторное прохождение.

Навыки форматирования: Реализован красивый табличный вывод результатов в консоли с выравниванием столбцов через string.Format.
